Posts are stored without sanitization and rendered as raw HTML. Inject a script tag to trigger on page load.

The /file endpoint joins paths without sanitization. Try traversing to read server files.
💻 Command Injection — Ping Critical
The ping endpoint passes user input directly to exec(). Chain system commands with ; or |.

User profiles are accessible by ID with no ownership check. Enumerate all users by changing the ID.
